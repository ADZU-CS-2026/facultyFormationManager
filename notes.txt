User (Person) Entity
├── id (primary key)
├── last_name
├── first_name
├── middle_initial
├── department_type (Admin, FFP, CON, CSITE, SED, SLA, SMA, CS, PPO)
├── position (nullable - only for Admin/CS)
├── office (nullable - only for Admin/CS)
├── status (nullable - only for PPO)
├── work_status (Active, Retired, Resigned, On Leave, Transferred)
├── work_status_date
├── work_status_notes
└── created_at, updated_at

Retreat_Record Entity
├── id (primary key)
├── user_id (foreign key)
├── retreat_type (DGY1, DGY2, DGY3, DGY4.1_IMC, DGY4.2_3D_Retreat, DGY5.1_IPC, DGY5.2_3D_Retreat, SD, 3D_Retreat)
├── school_year (e.g., "2023-2024")
├── completion_date
├── notes
└── created_at, updated_at

Certificate Entity
├── id (primary key)
├── retreat_record_id (foreign key)
├── certificate_number (unique - format: DGY2-2024-001)
├── issued_date
├── issued_by (foreign key to user who issued it)
├── status (Active, Reprinted, Voided)
├── print_count
├── void_reason
├── template_version
└── created_at
